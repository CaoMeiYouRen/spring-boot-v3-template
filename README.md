# spring-boot-v3-template

基于 Java 的 Spring Boot 3.X 项目模板

## 依赖要求

- Java >= 17
- Maven >= 3.6

## 安装依赖

```sh
mvn install
```

## 构建

```sh
mvn clean package -Dmaven.test.skip=true
```

## 运行

```sh
java -jar target/main.jar
```

## 测试

```sh
mvn test
```
